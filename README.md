# baseball_game(숫자 야구게임)

<h5>사용 스킬 : typescript, react, socket.io, recoil </p>

<p>설명 : 학생 때 하던, 숫자 야구 게임에서 착안하여 만들었습니다. </p>
<p>유저 2명이 socket.io로 통신하며, 게임을 진행합니다. 서로가 상대방의 숫자를 추리하여 맞추는 게임입니다. </p>
